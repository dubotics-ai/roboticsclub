# Rota `/vod/categories`

Assim como a rota [/tv/categories](/tv/categories), `/vod/categories` é um endpoint importante da API Quero Assistir, que fornece uma lista de categorias disponíveis para conteúdo sob demanda. As categorias são identificadas por IDs exclusivos e associadas a nomes descritivos que ajudam os usuários a encontrar o tipo de conteúdo que desejam assistir.
